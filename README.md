# Yelp Clone: Palatable

A clone of [Yelp](https://www.yelp.co.uk/london) built in Ruby on Rails.
This project was designed to introduce us to Rails, we spent 5 days building our app Palatable üç¥

## Technology

#### Framework
- Rails 4.2.8
- Devise for User Model
- OmniAuth for Facebook Sign In

#### Database
- Postgresql

#### Testing
- RSpec
- Capybara
